## Introducción a las imágenes satelitales

Son representaciones digitales (matrices de píxeles) de la superficie o atmósfera terrestre que son capturadas por satélites en órbita. Estas imágenes pueden ser utilizadas para una variedad de aplicaciones, incluyendo monitoreo ambiental, agricultura, urbanismo y gestión de desastres.

## Fundamentos

Las imágenes satelitales se componen de píxeles, cada uno representando un valor de reflectancia o radiancia en una banda espectral específica. Las bandas pueden ser visibles (RGB), infrarrojas, térmicas, entre otras.

## Fuentes de datos satelitales

- **Landsat**: Proporciona imágenes multiespectrales de alta resolución espacial y temporal.
- **Sentinel**: Parte del programa Copernicus de la UE, ofrece imágenes ópticas y radar.
- **MODIS**: Sensor a bordo de los satélites Terra y Aqua, proporciona datos diarios a nivel global.
- **Google Earth Engine**: Plataforma en la nube que permite el acceso y análisis de grandes conjuntos de datos satelitales.

Sobre el **PerúSat-1**, es un satélite de observación terrestre peruano que proporciona imágenes de alta resolución para aplicaciones en agricultura, gestión de recursos naturales y monitoreo ambiental.

## Herramientas para la visualización y análisis de imágenes satelitales

- **QGIS**: Software de código abierto para análisis geoespacial que soporta imágenes satelitales.
- **Google Earth Engine**: Plataforma en la nube para el análisis de grandes conjuntos de datos satelitales.
- **ArcGIS**: Software comercial para análisis geoespacial con capacidades avanzadas de procesamiento de imágenes.
- **Python con librerías como Rasterio, GDAL, y EarthPy**: Permite el procesamiento y análisis de imágenes satelitales mediante programación.

## Opciónes de descarga de imágenes satelitales

- **USGS Earth Explorer**: Plataforma para descargar imágenes Landsat y otros datos satelitales.

  Enlace: [USGS Earth Explorer](https://earthexplorer.usgs.gov/)

- **Copernicus Open Access Hub**: Portal para descargar imágenes Sentinel.  

  Enlace: [Copernicus Open Access Hub](https://browser.dataspace.copernicus.eu/)
